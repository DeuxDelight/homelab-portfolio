## Why This Repo Exists

This repository is intentionally **documentation-focused**.

Each lab includes:
- What I was trying to accomplish
- What broke
- Why it broke
- How I fixed it
- How I verified success

This mirrors real-world troubleshooting and change management, not theoretical exercises.
